# 📧 AI Email Subject Line Generator

An AI-powered tool that generates professional email subject 
lines using OpenAI's GPT model.

Built while following Ed Donner's LLM course.

---

## 🚀 What This Does

- Takes your email body as input
- Uses GPT-4.1-nano to analyze it  
- Generates 5 professional subject line suggestions instantly

---

## 🛠️ Tech Stack

- Python
- OpenAI API (GPT-4.1-nano)
- Google Colab (Jupyter Notebook)

---

## ⚙️ How to Run

### 1. Open in Google Colab
Click the "Open in Colab" badge above

### 2. Add your OpenAI API Key
- Click the 🔑 key icon in the left sidebar
- Add a secret named: OPENAI_API_KEY
- Paste your key as the value

### 3. Run All Cells
- Runtime → Run All
- That's it! ✅

---

## 📝 Example

**Input email:**
> "Hey George, find the quotation attached 
>  with modifications as per our discussion"

**Generated subject lines:**
1. Document Attached as Discussed
2. Follow-Up: Attached Document for Review
3. Your Requested Document is Attached
